# Create tree annotation for heatmap

Create tree annotation for heatmap

## Usage

``` r
create_tree_annotation(tree_ggplot, tree_width, n_cells)
```

## Arguments

- tree_ggplot:

  ggplot tree object

- tree_width:

  Width of tree annotation

- n_cells:

  Number of cells in heatmap

## Value

HeatmapAnnotation object
