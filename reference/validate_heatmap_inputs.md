# Validate input parameters for heatmap plotting

Validate input parameters for heatmap plotting

## Usage

``` r
validate_heatmap_inputs(data, tree, chromosomes_to_plot, to_plot)
```

## Arguments

- data:

  Input data frame

- tree:

  Phylogenetic tree object (optional)

- chromosomes_to_plot:

  Vector of chromosomes to include

- to_plot:

  Vector of features to plot
